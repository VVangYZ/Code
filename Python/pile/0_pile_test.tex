\section{桩长计算}%
\label{sec:}%
根据规范5.3.3可得，摩擦桩单桩承载力为%
\[%
[R_a] =\frac{1}{2}u\sum_{i=1}^nq_{ik}l_i+A_pq_r%
\]%

%
根据规范5.3.4可得，端承桩单桩承载力为%
\[%
[R_a]= c_1A_pf_{rk} +u\sum_{i=1}^mc_{2i}h_if_{rki} +\frac{1}{2}\xi_su\sum_{i=1}^nl_iq_{ik}%
\]%

%
考虑桩身自重与置换土重，桩基承载力为%
\[%
R_a =[R_a]-G_p+G_s%
\]%
